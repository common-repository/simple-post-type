== Description ==

Simple Post Type Allows you to add custom post for example: movies, music, reviews recommendations, support and more. In fact, you can create another type of posts under the name you want to select.
What you can do with Simple Post Type:

* Add Post Type the name and the link of your choice
* Separate your custom content from posts regular WordPress
* Add posts type into menus

== Installation ==

Upload the Simple Post Type plugin to your plugin folder on your site, Activate it, then enter Post Type Menu.
